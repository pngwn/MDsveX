Features

Composable

Build complex UIs from small, reusable pieces. Each component is designed to work seamlessly with others.

Accessible

WAI-ARIA compliant out of the box. Keyboard navigation, screen reader support, and focus management are built in.

Performant

Tree-shakeable, lazy-loadable, and optimized for minimal bundle size. Only ship what you import.

Themeable

CSS custom properties for easy theming. Supports dark mode, high contrast, and custom color schemes.
